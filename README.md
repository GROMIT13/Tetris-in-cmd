# Tetris-in-cmd

Recreation of modern tetris, displayed in console.

download [here](https://github.com/GROMIT13/Tetris-in-cmd/releases)

**Controlls**
```
|←| - move left
|→| - move right
|↓| - soft drop
|SPACE| - hard drop
|X| or |↑| - rotate clockwise
|Z| - rotate counterclockwise
|C| - hold tetromino
```
